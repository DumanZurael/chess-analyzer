# מנתח שחמט אישי – Chess Analyzer

אפליקציית web מקומית לניתוח משחקי שחמט שלך מ-chess.com, כולל מנוע **Stockfish** מובנה בדפדפן, ניווט בין מהלכים, סיווג איכות מהלך והצגת המהלך הטוב ביותר – ללא תשלום, ללא הרשמה, וללא שליחת נתונים החוצה.

## הפעלה ראשונה

1. ודא ש-XAMPP רץ ושירות **Apache** פעיל.
2. פתח דפדפן בכתובת: <http://localhost/ONE/chess-analyzer/>
3. הזן את שם המשתמש שלך מ-chess.com (כברירת מחדל: `dumantheking`) ולחץ **שמור**.
4. לחץ על **סרוק עכשיו** – האפליקציה תמשוך את 3 החודשים האחרונים של המשחקים שלך.
5. לחץ על כל משחק כדי לפתוח אותו, ולחץ **נתח משחק זה** כדי שהמנוע יסרוק אותו מהלך-מהלך.

## סריקה יומית אוטומטית

יש **שתי שכבות הגנה** שמבטיחות שלא תפספס משחקים:

### 1. בדיקה אוטומטית בכל פתיחה
בכל פעם שאתה פותח את האפליקציה, אם עברו יותר מ-24 שעות מהסריקה האחרונה – היא תרוץ ברקע ותעדכן את המשחקים החדשים שלך.

### 2. Windows Task Scheduler (יומי, אוטומטי לחלוטין)
כדי שהסריקה תרוץ גם בלי שתפתח את האפליקציה ידנית:

1. פתח את **Task Scheduler** (לחץ על מקש Windows וחפש "Task Scheduler").
2. בחר **Create Basic Task…**
3. שם: `Chess Analyzer Daily Scan`
4. **Trigger**: Daily, בחר שעה (לדוגמה: 09:00 בבוקר).
5. **Action**: Start a program
6. בחלון Program/script לחץ Browse ובחר את הקובץ:
   ```
   D:\XAMPP\htdocs\ONE\chess-analyzer\scan-daily.bat
   ```
7. **Finish**.
8. ודא שב-XAMPP Apache רץ אוטומטית עם הפעלת המחשב (אפשר להגדיר ב-XAMPP Control Panel → Config → Service and Port Settings).

## מה האפליקציה עושה?

### מסך ראשי
- שולפת את כל המשחקים שלך מ-chess.com Public API (חינמי, לא דורש מפתח).
- מציגה רשימה של כל המשחקים: תאריך, צבע, יריב, תוצאה, סטטוס ניתוח.
- חמש סטטיסטיקות כלליות: סה"כ, נותחו, ניצחונות, הפסדים, דיוק ממוצע.

### מסך ניתוח משחק
- **לוח שחמט** מלא עם הכלים, עם אפשרות להפוך אותו (כפתור ⇅).
- **פס הערכה** מימין שמראה מי מוביל ובכמה (כמו ב-chess.com).
- **רשימת מהלכים** מימין, עם נקודה צבעונית לכל מהלך שמסמלת את האיכות:
  - ★ ירוק – הטוב ביותר
  - ✓ – טוב/מצוין
  - 📖 חום – מהלך פתיחה
  - ?! צהוב – אי-דיוק
  - ? כתום – טעות
  - ?? אדום – פאדיחה
- **חץ ירוק על הלוח** שמראה איזה מהלך **היה צריך** לבצע (כשעשית טעות).
- **תיבת ניתוח** עם הסבר על המהלך הנוכחי.
- **ניווט**:
  - חץ שמאלה במקלדת – מהלך קדימה
  - חץ ימינה – מהלך אחורה
  - Home – חזרה להתחלה
  - End – מעבר לסוף
  - F – הפוך לוח

## עומק ניתוח

יש בחירה של עומק (כמה מהלכים קדימה המנוע "רואה"):
| עומק | מהירות | חוזק |
|------|--------|-----|
| 12 | מהיר מאוד (~1 שנייה למהלך) | בערך 2000 ELO |
| 15 | מאוזן (~2 שניות) | בערך 2400 ELO |
| 18 | איטי (~5 שניות) | בערך 2800 ELO |
| 20 | מאוד איטי (~10 שניות) | חזק יותר מבני אדם |

ניתוח של משחק 30-מהלכים בעומק 15 לוקח כדקה. עומק 18-20 ייתן ניתוח קרוב יותר ל-chess.com Premium אבל ייקח 3-5 דקות למשחק.

## אחסון

כל הנתונים נשמרים ב-**localStorage של הדפדפן** (~5-10MB מקסימום).
אין שליחה חיצונית. אם תרצה לאפס – פתח DevTools (F12) → Application → Local Storage → ימני → Clear.

## טכנולוגיות

- **HTML + CSS + Vanilla JS** – ללא build step, רץ ישירות תחת XAMPP.
- **chess.js** – ספרייה ללוגיקת שחמט (PGN, הלוח, מהלכים חוקיים).
- **Stockfish.js v11** – מנוע השחמט החזק בעולם, גרסת JavaScript שרצה כ-Web Worker בדפדפן.
- **chess.com Public API** – לשליפת המשחקים.

## פתרון בעיות

- **"משתמש לא נמצא"** – ודא ששם המשתמש מדויק (אותיות קטנות) ושיש לפחות משחק ציבורי אחד בפרופיל.
- **"Stockfish לא הגיב"** – ודא שיש אינטרנט (המנוע נטען מ-CDN בפעם הראשונה). אחרי טעינה ראשונה הדפדפן עושה קאש.
- **לוח לא נראה כמו שצריך** – לחץ Ctrl+F5 לרענון מלא ועקיפת הקאש.
- **הסריקה מאוד איטית** – chess.com מגביל קצב בקשות. סריקה אחת לשבוע של 3 חודשים אחרונים זה גודל סביר.
